# 🐍 Змейка — Классика с расширенной кастомизацией

[![Python](https://img.shields.io/badge/Python-3.7+-blue.svg)](https://www.python.org/)
[![Pygame](https://img.shields.io/badge/Pygame-2.0+-green.svg)](https://www.pygame.org/)
[![License](https://img.shields.io/badge/License-GPLv3-red.svg)](https://www.gnu.org/licenses/gpl-3.0.html)

Классическая игра «Змейка» с возможностью полной настройки внешнего вида. Игра выходит за рамки стандартной реализации благодаря гибкой системе изменения цветов, декоративных элементов и сохранению пользовательских предпочтений.


## ✨ Особенности

- **🎮 Классический геймплей**: Управляй змейкой, собирай еду, избегай столкновений со стенами и собственным хвостом.
- **🎨 Глубокая кастомизация**:
  - Изменяй **цвета** всех игровых элементов: голова, тело, еда, фон и сетка.
  - Выбирай **шляпу** для змейки (Цилиндр, Треугольная, С полями, Корона, Кепка и другие).
  - Добавляй **узор** на тело змейки (Полоски, Точки, Кресты, Шахматы).
- **💾 Сохранение настроек**: Все изменения автоматически сохраняются в файл `settings.json` и загружаются при следующем запуске.
- **🖥️ Интуитивный интерфейс**: Главное меню, меню настроек с вкладками и живое превью изменений в реальном времени.

## 🧩 Архитектура и система настроек

### Модули кастомизации

| Компонент | Реализация |
| :--- | :--- |
| **Цветовая палитра** | Словарь `COLORS` содержит все цвета игры. Пользователь изменяет их через RGB-слайдеры, значения сохраняются в JSON. |
| **Система шляп** | Массив `HAT_COLORS` хранит цвета для каждой шляпы. Выбор шляпы сохраняется как индекс (`SELECTED_HAT`) в `settings.json`. |
| **Паттерны узоров** | Матрица `PATTERNS` определяет алгоритмы отрисовки узоров на теле змейки. Каждый узор — это функция, применяемая к каждому сегменту. |
| **Персистентность** | Функции `load_settings()` и `save_settings()` обеспечивают загрузку и сохранение настроек между сессиями. |
| **Динамическая скорость** | Скорость игры увеличивается по мере набора очков: `speed = 10 + score // 5`. |

### Структура данных

Все настройки хранятся в едином JSON-файле:

```json
{
  "colors": {
    "snake_head": [0, 200, 0],
    "snake_body": [0, 255, 0],
    "food": [255, 50, 50],
    "background": [20, 20, 30],
    "grid": [50, 50, 60]
  },
  "hat": 2,
  "pattern": 1
}
```
